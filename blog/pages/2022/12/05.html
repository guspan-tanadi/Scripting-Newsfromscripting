feedToMasto is a Node app that periodically checks a list of feeds, posting new items to Mastodon. Good example code to get started with the Mastodon API. Also illustrates how easy feeds (RSS, Atom, RDF) can be incorporated into any Node app with the reallySimple package. Finally, I hope to start building a rep in the Mastodon development community with this package. I hope to make quite a few contributions in the future. If you have questions here's a good place to start. #
